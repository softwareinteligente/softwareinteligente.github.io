Ecuapass-docs: Suite de Manejo Inteligente de Documentos en el ECUAPASS

La Suite de Manejo Inteligente de Documentos en el Sistema de Aduanas del Ecuador “ECUAPASS”, es una solución integral e innovadora diseñada para agilizar y optimizar la gestión de los documentos necesarios para el transporte terreste internacional: cartas de porte (CPIC), manifiestos de carga (MCI), y declaraciones de tránsito (DTAI).

La suite cuenta con cinco módulos:

  • Módulo de elaboración de documentos: permite crear los documentos de forma rápida y precisa, gracias a un auto-completado inteligente. Mire este video en YouTube

  • Módulo de almacenamiento de información: permite guardar de forma automática en una base de datos la información principal de los documentos y de las entidades relacionadas en estos (ej. empresas, vehículos, y conductores).

  • Módulo de digitación automática de cartas de porte (CPIT): permite digitar automáticamente las cartas de porte al sistema ECUAPASS. Mire este vide en YouTube

  • Módulo de digitación automática de manifiestos (MCI): permite digitar automáticamente los manifiestos de carga al sistema ECUAPASS. Mire este vide en YouTube

  • Módulo de digitación automática de declaraciones (DTAI): permite digitar automáticamente las declaraciones de tŕansito al sistema ECUAPASS. Mire este vide en YouTube

Los tres módulos de digitación automática extraen la información de sus respectivos documentos y por medio de un bot o robot de software, digitan automáticamente está información en los sistemas respectivos del ECUAPASS, y esto se realiza en cuestión de segundos y de forma segura.

Módulo 1: Creación de Documentos

  • Permite la elaboración de documentos de manera eficiente.
  • Auto completa muchos de los campos, reduciendo errores y tiempos de digitación.
  • Almacena automáticamente la información en la base de datos para luego usarse para auto-completar.
  • Presenta una fácil interacción con el usuario, como si estuviera escribiendo sobre el documento físico.

Módulo 3: Digitación Automática de Cartas de Porte (CPIT) al Sistema ECUAPASS

  • Extrae todos los datos digitados en la carta de porte.
  • Analiza la información y extrae los datos necesarios para digitar en el sistema ECUAPASS.
  • Digita automáticamente la información al sistema de carta de portes del sistema ECUAPASS.
  • Guarda la información de la carta de porte en la base de datos
  • Almacena o actualiza la base de datos con los datos de empresas mencionados en la carta de porte.

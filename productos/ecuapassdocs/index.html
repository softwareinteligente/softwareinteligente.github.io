Creación Inteligente de Documentos del ECUAPASS

EcuapassDocs

Permite manejar y crear de forma simple y semi-automática los tres tipos de documentos requeridos para el Transporte Terrestre de Carga Internacional en el sistema de aduanas del Ecuador “ECUAPASS”: Cartas de Porte, Manifiestos, y Declaraciones.

Características

  • Aplicación web: Permite crear, consultar, y modificar en línea los tres tipos de documentos:

  • Completado semi-automático: De forma “inteligente” la aplicación auto-completa los datos de los documentos, agilizando su creación y reduciendo la digitación manual

  • Almacenamiento de datos completo: Toda la información se guarda en una base de datos en la web. Se almacena tanto los documentos, como los datos que ellos contienen tales como empresas, conductores, y vehículos. Y todo esto lo hace automáticamente, es decir, a medida que se crea el documento, la aplicación extrae la información correspondiente y la almacen en la base de datos.

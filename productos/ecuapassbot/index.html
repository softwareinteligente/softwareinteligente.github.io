Ecuapass-Bot

Transmisión automática de documentos al ECUAPASS

EcuapassBot

Ecuapass-Bot es una herramienta novedosa que transmite (digita) automáticamente los documentos de Transporte Terreste Internacional (Cartas de Porte, Manifiestos) al sistema ECUAPASS.

Carta de Porte

Manifiesto

Características

  • Extracción Inteligente de Información: Toma la información de sus documentos PDF o directamente de su sistema de creación de documentos y extrae de forma inteligente todos los datos requeridos en el sistema ECUAPASS.

  • Pre-chequeo de los datos: Los datos extraidos se presentan en una aplicación similar al sistema ECUAPASS para que el usuario los verifique y corriga, si es necesario, antes de transmitirlos.

  • Transmisión automática: Un robot de software digita estos datos directamente en el formulario respectivo del sistema ECUAPASS. Finalmente, el usuario los revisa y hace el envío final.

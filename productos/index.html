Nuestros Productos

Software para la optimización y automatización de procesos

Software orientado al manejo de documentos de transporte terrestre internacional, tanto para:

  • Elaboración semi-automática de Cartas de Porte, Manifiestos, y Declaraciones de Tránsito Aduanero
  • Transmisión automática de estos documentos al sistema aduanero ecuatoriano ECUAPASS.

EcuapassBot

Transmite (digitar automáticamente) los documentos al sistema ECUAPASS de manera rápida (en cuestión de segundos) y segura (sin errores de transcripción).

EcuapassDocs

Permite la creación de Cartas de Porte, Manifiestos y Declaraciones de Importación. Los datos de cada casilla se autocompletan inteligentemente y se guarda automáticamente en Bases de Datos toda la información que se derive de estos datos (empresas, vehículos, conductores, etc.)

EcuapassSuite

Integra en una sola aplicación las aplicaciones anteriores: creación con EcuapassDocs y trasmisión con EcuapassBot
